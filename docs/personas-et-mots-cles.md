## 0. Le diagnostic — pourquoi la campagne du brief initial aurait tourné à vide

Les 12 mots-clés de la §4 de `acquisition-payante.md` ont été testés sur données
réelles. **11 sur 12 ont un volume de recherche non mesurable** (< ~10/mois) au
Québec comme au Canada FR. Seul « automatisation facturation » a du volume
(~20/mois), avec un CPC de **74 $** et une concurrence HIGH au Québec.

La cause n'est **pas le sujet** (les CPC élevés prouvent que des annonceurs paient
cher pour ce marché), c'est **la formulation**. Le message du site est bâti sur une
douleur *invisible et latente* — « Vos journées disparaissent dans des tâches
qu'une machine ferait sans se tromper. » C'est excellent pour *closer* et pour
Meta, mais **personne ne tape ça dans Google**. Les gens qui ont cette douleur
cherchent la **catégorie de solution** une fois qu'ils l'ont nommée eux-mêmes
(« logiciel de gestion X », « logiciel sur mesure », « rpa »), ou le nom du
**métier** qu'ils exercent.

**Conclusion : le message est au mauvais endroit du funnel, et le ciblage est trop
abstrait.** La correction = nommer des verticales.

**Arrêter de vendre « l'automatisation » en abstrait. Nommer des métiers.**
Une page = un persona = un mot-clé « logiciel gestion [métier] » + le cas 56 000 $

## 3. Les personas ultra-nichés (le cœur)

Classés par **proximité avec le cas prouvé** (épicerie indépendante = synchro de
données ↔ inventaire ↔ prix). Plus un persona est proche, plus le pitch est
crédible. Chaque persona = un secteur précis + un job cherchable + l'outil qu'ils
fuient + pourquoi Gabriel gagne.

### 🎯 Tier 1 — le « bullseye » (calqué sur l'épicerie : données / inventaire / prix)

**1. Épicerie / dépanneur indépendant** *(ton cas prouvé)*
- **Décideur** : propriétaire. **Taille** : 1-3 succursales.
- **Déclencheur** : marges qui fondent, prix fournisseurs jamais à jour, inventaire faux.
- **Job** : synchroniser prix fournisseurs ↔ caisse, suivre l'inventaire réel.
- **Fuient** : Excel + caisse déconnectée + factures fournisseurs papier.
- **Pourquoi toi** : *tu l'as déjà fait — 56 000 $/an récupérés.* Crédibilité maximale.
- **Mots-clés** : `logiciel gestion épicerie`, `gestion prix fournisseurs`, `logiciel gestion inventaire`, `logiciel point de vente`.

**2. Distributeur / grossiste alimentaire régional**
- **Décideur** : proprio / directeur des opérations. **Taille** : 5-40 employés.
- **Déclencheur** : commandes reçues par courriel/fax retapées à la main, catalogue de prix ingérable.
- **Job** : automatiser la prise de commande, tenir un catalogue de prix par client, planifier les tournées.
- **Fuient** : Excel + fax + double saisie commandes → facturation.
- **Pourquoi toi** : même ADN que l'épicerie (synchro prix/commandes), un cran plus haut.
- **Mots-clés** : `logiciel gestion de commande` (comp. 0 !), `logiciel distribution`, `logiciel wms`, `logiciel gestion des stocks`.

**3. Quincaillerie / détaillant indépendant** *(non-alimentaire)*
- **Décideur** : propriétaire. **Taille** : 1-2 magasins.
- **Déclencheur** : inventaire multi-fournisseurs impossible à suivre, ruptures et surstocks.
- **Job** : inventaire unifié, réapprovisionnement, prix par fournisseur.
- **Fuient** : Excel + POS basique qui ne parle pas à l'inventaire.
- **Pourquoi toi** : synchro inventaire = ton terrain exact.
- **Mots-clés** : `logiciel inventaire`, `logiciel gestion inventaire`, `logiciel point de vente`.

**4. Petit manufacturier / atelier de production**
- **Décideur** : proprio / contremaître. **Taille** : 5-50 employés.
- **Déclencheur** : bons de travail sur papier, on ne sait jamais le coût réel d'un lot.
- **Job** : suivi de production, bons de travail, inventaire matières premières.
- **Fuient** : Excel + papier à l'atelier + « la vraie donnée est dans la tête du contremaître ».
- **Pourquoi toi** : chiffrer une tâche invisible et la systématiser = ta signature.
- **Mots-clés** : `gpao`, `erp manufacturier`, `logiciel gmao`, `logiciel gestion production`.

**5. Grossiste / distributeur de pièces B2B** *(auto, industriel, plomberie)*
- **Décideur** : propriétaire. **Taille** : 5-30 employés.
- **Déclencheur** : catalogue de milliers de références, prix négociés par client, commandes récurrentes.
- **Job** : catalogue + prix par client + commandes EDI/portail.
- **Fuient** : Excel + vieux ERP maison que « plus personne n'ose toucher ».
- **Pourquoi toi** : refonte par tranches d'un vieux système = ton offre refonte.
- **Mots-clés** : `logiciel gestion commandes`, `logiciel wms`, `solution erp`, `intégrateur erp`.

### 🔧 Tier 2 — projets & temps (douleur ops forte, angle « soumission / chantier / route »)

**6. Entrepreneur général / construction** *(fort volume : « gestion de projet construction » = 210/mois)*
- **Décideur** : proprio. **Job** : soumissions, feuilles de temps chantier, suivi des coûts par projet.
- **Fuient** : Excel + paperasse chantier + factures qui traînent.
- **Pourquoi toi** : le suivi de coût par projet est un problème de *données entre systèmes* — ton créneau.
- **Mots-clés** : `gestion de projet construction`, `logiciel soumission construction`, `logiciel feuille de temps`.

**7. Sous-traitant spécialisé** *(électricien, plombier, CVC, réfrigération)*
- **Décideur** : proprio. **Job** : cédule des techniciens, bons de travail, facturation terrain.
- **Fuient** : agenda papier + Excel + appels pour répartir.
- **Pourquoi toi** : automatiser cédule → bon de travail → facture = un sprint idéal.
- **Mots-clés** : `logiciel gestion électricien/plombier`, `logiciel bon de travail`, `logiciel feuille de temps`.

**8. Transporteur / entreprise de livraison régionale**
- **Décideur** : proprio / répartiteur. **Job** : répartition, feuilles de route, facturation au voyage.
- **Fuient** : Excel + téléphone + tableau blanc.
- **Pourquoi toi** : synchro données (route → facturation) = ton ADN.
- **Mots-clés** : `logiciel gestion transport`, `logiciel répartition`, `logiciel logistique`.

**9. Traiteur / boulangerie / production alimentaire artisanale**
- **Décideur** : proprio. **Job** : commandes, planning de production, coût de recette, étiquetage.
- **Fuient** : Excel + cahier + « on calcule les coûts à l'instinct ».
- **Pourquoi toi** : angle alimentaire = prolongement direct de l'épicerie.
- **Mots-clés** : `logiciel gestion traiteur`, `logiciel production alimentaire`, `logiciel restaurant`.

### 🗓️ Tier 3 — RDV & dossiers (angle différent, mais « logiciel prise de rendez-vous » a du volume)

**10. Clinique privée** *(dentaire, physio, véto, esthétique, podiatrie)*
- **Décideur** : proprio / gestionnaire. **Job** : RDV, rappels, dossiers, facturation.
- **Fuient** : agenda papier + vieux logiciel qui ne rappelle pas les clients.
- **Pourquoi toi** : automatiser les rappels = ROI mesurable (no-shows évités).
- **Mots-clés** : `logiciel gestion clinique`, `logiciel prise de rendez-vous`, `logiciel gestion patients`.

**11. Garage / atelier mécanique**
- **Décideur** : proprio. **Job** : bons de travail, inventaire pièces, rappels d'entretien.
- **Fuient** : Excel + papier + logiciel de pièces déconnecté de la facturation.
- **Mots-clés** : `logiciel gestion garage`, `logiciel atelier mécanique`, `logiciel bon de travail`.

**12. Gestion immobilière / locatif** *(« logiciel gestion immobiliere » = 50/mois)*
- **Décideur** : proprio / gestionnaire. **Job** : baux, loyers, entretien, avis aux locataires.
- **Fuient** : Excel + classeurs + relances manuelles.
- **Mots-clés** : `logiciel gestion immobilière`, `logiciel gestion locative`.

## 4. Structure de campagne recommandée (remplace la §4 du brief)

**Google Search — 3 groupes d'annonces sur les pépites commerciales :**
- **G1 · Sur-mesure / dev** : `logiciel sur mesure`, `développement logiciel sur mesure`, `développeur logiciel`, `développement web sur mesure`. → landing `/diagnostic/` ou page « logiciel sur mesure ».
- **G2 · Remplacer un système** : `solution erp`, `progiciel de gestion intégré`, `intégrateur erp`. → landing angle refonte.
- **G3 · Inventaire / gestion** : `logiciel gestion inventaire`, `logiciel gestion de stock`, `solutions crm`, `crm client`. → landing angle « sortir d'Excel ».
- **Négatifs** : `emploi`, `gratuit`, `formation`, `cours`, `salaire`, `télécharger`, `open source`, `crack`, `pdf`.
- Budget 30-50 $/j, *Maximiser les conversions* → *tCPA* après ~15-20 conversions.

**SEO organique — piliers + pages verticales :**
- 3-4 articles piliers : `rpa`, `automatisation`, `progiciel vs sur-mesure`, définitions ERP.
- 1 page service par persona Tier 1 (5 pages), déclinaison du cas 56 000 $.
- Maillage : piliers → pages verticales → `/diagnostic/`.

**Meta / LinkedIn — le message latent, phase 1 (pas phase 2) :**
- La demande de recherche pour la douleur latente est mince → c'est Meta/LinkedIn
  qui doit la générer, par ciblage **poste + secteur** (pas mot-clé).
- Ton message actuel (« vos journées disparaissent… ») est parfait ici, avec le
  **nom du métier dans le visuel** (« Vous gérez une quincaillerie ? »).
